## ‚óą What is Stitch?

Stitch bridges the gap between raw, messy data and actionable intelligence. Upload any CSV, Excel, or JSON file and immediately access a full suite of tools ‚ÄĒ from one-click data cleaning to natural language querying to automated machine learning ‚ÄĒ all powered by Gemini AI at the reasoning layer and trusted Python libraries at the compute layer.

**The core philosophy:** LLMs reason, Python crunches. Your raw data never leaves your session.

## ‚óą Quick Start

### 1 ¬∑ Clone the repository

```bash
git clone https://github.com/yourusername/stitch-ai-data-suite.git
cd stitch-ai-data-suite
```

### 2 ¬∑ Create a virtual environment

```bash
python -m venv .venv

# macOS / Linux
source .venv/bin/activate

# Windows
.venv\Scripts\activate
```

### 3 ¬∑ Install dependencies

```bash
pip install -r requirements.txt
```

### 4 ¬∑ Configure your API key

```bash
cp .env.example .env
# Open .env and add your Gemini key:
# GEMINI_API_KEY=AIzaSy...your-key-here
```

> Get a free key at [aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey) ‚ÄĒ no credit card required.

### 5 ¬∑ Launch

```bash
streamlit run app.py
```

Open **[http://localhost:8501](http://localhost:8501)** in your browser.

## ‚óą Gemini API Setup

1. Visit [https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)
2. Sign in with your Google account
3. Click **Create API Key**
4. Copy the generated key
5. Paste it into your `.env` file:
   ```
   GEMINI_API_KEY=AIzaSy...your-key-here
   ```
6. Restart the app

> **Note:** AI features are fully optional. Stitch works without an API key ‚ÄĒ only the AI Insights page, Chat narration, and AutoML interpretation require Gemini.

## ‚óą Security & Privacy

| Concern | How Stitch handles it |
|---|---|
| **Session isolation** | Each browser session gets a unique UUID ‚ÄĒ no shared memory between users |
| **API key safety** | Loaded via `.env` locally; Streamlit Secrets in production ‚ÄĒ never hardcoded |
| **LLM data exposure** | Only schema + aggregated stats + ‚Č§ 5 masked rows are sent to Gemini |
| **PII detection** | Regex-based scan covers email, phone, SSN, Aadhaar, PAN, credit cards |
| **Temporary files** | `tempfile` module ‚ÄĒ all session files deleted after processing |
| **Error handling** | Generic messages to users; full stack traces only in server logs |

## ‚óą Troubleshooting

| Symptom | Fix |
|---|---|
| `ModuleNotFoundError` | Run `pip install -r requirements.txt` |
| AI features not working | Verify `GEMINI_API_KEY` is set in `.env` or Streamlit Secrets |
| Excel file not parsing | Confirm `openpyxl` is installed: `pip install openpyxl` |
| PDF not generating | Confirm `reportlab` is installed: `pip install reportlab` |
| XGBoost not available | `pip install xgboost` ‚ÄĒ optional, app falls back to Random Forest + Logistic Regression |
| Large files running slowly | Reduce dataset size, or raise the Streamlit upload limit in `.streamlit/config.toml` |
